# MentalGuard-AI

**Mental Health Monitoring System** using Camera, Voice & Simulated Sensors.

A responsible Python ML project that collects data over multiple sessions before giving mental health insights.

## Features
- Real-time face emotion detection
- Voice sentiment analysis
- Simulated biometric sensors
- Delayed verification (minimum 4 sessions)
- Beginner-friendly GUI

## Installation
```bash
pip install -r requirements.txt
python main.py
```

## How it Works
The system waits for enough data before making conclusions for accuracy.
